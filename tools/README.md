# tools/ - 工具与脚本文件夹

存放抓取脚本、数据处理脚本、格式化工具等。

## 用途

- 网页抓取脚本（Python/Node.js）
- 数据清洗与转换脚本
- 日报格式化工具（Markdown/HTML/PDF 生成）
- 辅助工具函数

## 结构示例

```
tools/
├── scraper.py          # 核心抓取脚本
├── processor.py        # 数据清洗处理
├── report_generator.py # 日报生成器
├── utils.py            # 通用工具函数
└── requirements.txt    # Python 依赖
```
